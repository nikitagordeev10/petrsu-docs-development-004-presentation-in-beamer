\frametitle{Решение примера }
Решение: Подставляя $x=2$ в выражение, получаем:

$$\frac{5x^2}{(x+1)^2}=\frac{5\cdot 2^2}{(2+1)^2}=\frac{20}{9}$$

Ответ: $\frac{5x^2}{(x+1)^2}$ при $x=2$ равно $\frac{20}{9}$.
